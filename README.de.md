## 🚀 Schnellstart

### Voraussetzungen

- **Node.js** 18.0+ 
- **npm** 7.0+ oder **yarn** 1.22+ oder **pnpm** 8.0+
- **Git** für Versionskontrolle

### Installation

```bash
# Repository klonen
git clone https://github.com/sxyseo/geminicli.cc.git
cd geminicli.cc

# Abhängigkeiten installieren
npm install
# oder
yarn install
# oder
pnpm install

# Entwicklungsserver starten
npm run dev
# oder
yarn dev
# oder
pnpm dev
```

### Umgebungseinrichtung

Erstellen Sie eine `.env.local` Datei im Hauptverzeichnis:

```env
# Optional: Analytics und Tracking
NEXT_PUBLIC_GA_ID=your_google_analytics_id
NEXT_PUBLIC_VERCEL_ANALYTICS=true

# Optional: Feature-Flags
NEXT_PUBLIC_FEATURE_PLAYGROUND=true
NEXT_PUBLIC_FEATURE_COMMUNITY=true
```

### **Entwicklungs-Workflow**

1. **Repository forken**
2. **Feature-Branch erstellen**
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. **Änderungen vornehmen**
4. **Tests und Linting ausführen**
   ```bash
   npm run lint
   npm run type-check
   npm run build
   ```
5. **Änderungen committen**
   ```bash
   git commit -m "Erstaunliches Feature hinzufügen"
   ```
6. **Zu Ihrem Fork pushen**
   ```bash
   git push origin feature/amazing-feature
   ```
7. **Pull Request erstellen**
